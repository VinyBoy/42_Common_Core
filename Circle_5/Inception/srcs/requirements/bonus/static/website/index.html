Projets

Minishell (C / UNIX)

Développement d’un mini shell en C reproduisant le comportement d’un terminal UNIX : parsing avancé des commandes, gestion des pipes, redirections, variables d’environnement, exit codes et signaux.

Objectif : comprendre en profondeur l’exécution de processus (fork/exec), la gestion des fichiers et le fonctionnement interne d’un shell.

Stack utilisée

  • C
  • UNIX
  • Linux
  • Make
  • Git

Webserv (C++ / HTTP)

Développement d’un serveur HTTP en C++ (C++98) inspiré de Nginx : gestion des sockets, multiplexing (select/poll/epoll selon l’implémentation), parsing de requêtes, routage, réponses, erreurs et configuration via fichier.

Objectif : comprendre les bases du réseau (TCP), le modèle événementiel, les standards HTTP, et construire une application performante et robuste côté serveur.

Stack utilisée

  • C
  • C++
  • UNIX
  • Linux
  • Make
  • Git

Inception (Docker / DevOps)

Mise en place d’une infrastructure complète avec Docker Compose : Nginx (TLS), WordPress, MariaDB, volumes persistants, secrets et réseau isolé, avec des images construites via Dockerfiles et une configuration reproductible.

Objectif : maîtriser la containerisation, l’automatisation d’environnements, la gestion de services en production et les bonnes pratiques DevOps.

Cub3D (C / Raycasting)

Création d’un moteur 3D “à la Wolfenstein” en C avec raycasting : lecture et validation de map, rendu en temps réel, textures, gestion de la caméra et des déplacements, collisions et interactions.

Objectif : comprendre les fondamentaux du rendu 2.5D, la gestion d’événements, l’optimisation et la structure d’un projet graphique en bas niveau.
